# Settings

## Change Password 

### 1. Open Settings

<div class="flex flex-col md:flex-row items-center gap-6">
  <div class="flex-1 text-gray-700 leading-relaxed">
    Click <strong>Settings</strong> from the main menu.
  </div>

  <img 
    src="https://res.cloudinary.com/dasd7oi7y/image/upload/v1772291988/vnztolq7itefcopmwrj4.webp"

</div>

### 2. Select Change Password

<div class="flex flex-col md:flex-row items-center gap-6">
  <div class="flex-1 text-gray-700 leading-relaxed">
    Click <strong>Change Password</strong> to update your account credentials.
  </div>

  <img 
    src="https://res.cloudinary.com/dasd7oi7y/image/upload/v1772291997/rjdixmghymyb3inszebu.webp" 
    alt="Change Password Option"
    class="w-[170px] h-auto rounded-2xl shadow-md"
  />
</div>

---

### 3. Enter Password Details

<div class="flex flex-col md:flex-row items-center gap-6">
  <div class="flex-1 text-gray-700 leading-relaxed">
    Enter your <strong>Current Password</strong>, then input <strong>New Password</strong> and <strong>Confirm Password</strong>.
  </div>

  <img 
    src="https://res.cloudinary.com/dasd7oi7y/image/upload/v1772292004/vacn2fhmzbmosfw0ee2i.webp" 
    alt="Enter Password Details"
    class="w-[170px] h-auto rounded-2xl shadow-md"
  />
</div>

---

### 4. Save Changes

<div class="flex flex-col md:flex-row items-center gap-6">
  <div class="flex-1 text-gray-700 leading-relaxed">
    Click <strong>Save</strong> to apply the new password.
  </div>

  <img 
    src="https://res.cloudinary.com/dasd7oi7y/image/upload/v1772292011/cg5psprl5exb5pnbkx8o.webp"

  />
</div>
