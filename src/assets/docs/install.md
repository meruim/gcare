## Install GCare App

### 1. Open the Downloaded File

<div class="flex flex-col md:flex-row items-center gap-6">
  <div class="flex-1 text-gray-700 leading-relaxed">
    Open the downloaded file to begin installation. (Skip this step if you have already installed the app.)
  </div>

  <img 
    src="https://res.cloudinary.com/dasd7oi7y/image/upload/v1772334697/67cf994c-7766-490c-8166-6cff30b8d788.webp" 
    alt="Open Downloaded File"
    class="w-full max-w-[220px] md:max-w-[260px] rounded-2xl shadow-md"
  />
</div>

---

### 2. Tap Settings

<div class="flex flex-col md:flex-row items-center gap-6">
  <div class="flex-1 text-gray-700 leading-relaxed">
    If prompted, tap <strong>Settings</strong>.
  </div>

  <img 
    src="https://res.cloudinary.com/dasd7oi7y/image/upload/v1772292149/s7pd5s8xmjtjxcap9gsz.webp" 
    alt="Settings Prompt"
    class="w-full max-w-[220px] md:max-w-[260px] rounded-2xl shadow-md"
  />
</div>

---

### 3. Turn on Allow from this Source

<div class="flex flex-col md:flex-row items-center gap-6">
  <div class="flex-1 text-gray-700 leading-relaxed">
    Turn on <strong>Allow from this source</strong>, then go back to continue the installation.
  </div>

  <img 
    src="https://res.cloudinary.com/dasd7oi7y/image/upload/v1772292091/qux0icavhi1jibtcdgrd.webp" 
    alt="Allow From This Source"
    class="w-full max-w-[220px] md:max-w-[260px] rounded-2xl shadow-md"
  />
</div>

---

### 4. Tap Install

<div class="flex flex-col md:flex-row items-center gap-6">
  <div class="flex-1 text-gray-700 leading-relaxed">
    Tap <strong>Install</strong> to begin installing the GCare app.
  </div>

  <img 
    src="https://res.cloudinary.com/dasd7oi7y/image/upload/v1772292112/flzxw7caedmqesm1vlq7.webp" 
    alt="Install App"
    class="w-full max-w-[220px] md:max-w-[260px] rounded-2xl shadow-md"
  />
</div>

---

### 5. Tap Open

<div class="flex flex-col md:flex-row items-center gap-6">
  <div class="flex-1 text-gray-700 leading-relaxed">
    After installation is complete, tap <strong>Open</strong> to launch the GCare app.
  </div>

  <img 
    src="https://res.cloudinary.com/dasd7oi7y/image/upload/v1772292134/d7viho8r1frg0x6abqmk.webp" 
    alt="Open App"
    class="w-full max-w-[220px] md:max-w-[260px] rounded-2xl shadow-md"
  />
</div>
